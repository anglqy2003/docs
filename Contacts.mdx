# Contacts

The **Contacts** section allows you to associate people—such as brokers, agents, or clients—with an entity like an Insured, Policy, or Claim. This improves clarity and ensures easy access to key contact details throughout your workflow.

---

## <span style={{ color: "#714CC5" }}>Accessing the Contacts Tab</span>

1. To create a contact, navigate to the **Insured** page by clicking the **Insured** tab on the sidebar.

   <img
     src="/images/Screenshot2025-07-14at2.45.05PM.png"
     alt="Screenshot2025 07 14at2 45 05PM Pn"
     title="Screenshot2025 07 14at2 45 05PM Pn"
     style={{ width:"20%" }}
   />
2. Click the entity you would like to edit (e.g. Insured, Policy, or Claim).
3. After that, from the navigation tabs near the top of the page, click **Contacts**.

  src="/images/Screenshot2025-07-14at2.50.20PM.png"

This opens a table view showing all existing contacts. If none have been added yet, you'll see a message:\
“No contacts have been added yet.”

---

## Adding a New Contact

To add a contact to the current entity:

1. Click the **Add Contact** button on the right side of the Contacts table.
   - This will open a dropdown with options to either:
     - Add New Contact
     - Link Existing Contact (if available)

   <img
     src="/images/Screenshot2025-07-14at2.52.38PM.png"
     alt="Screenshot2025 07 14at2 52 38PM Pn"
     title="Screenshot2025 07 14at2 52 38PM Pn"
     style={{ width:"77%" }}
   />
2. Select **Add New Contact**.
3. Fill in the contact details in the form:
   - Contact Name
   - Title
   - Email
   - Phone Number
   - Address
   - Apartment or Suite Number
   - Description (optional notes or context)

   <img
     src="/images/Screenshot2025-07-14at3.01.54PM.png"
     alt="Screenshot2025 07 14at3 01 54PM Pn"
     title="Screenshot2025 07 14at3 01 54PM Pn"
     style={{ width:"59%" }}
   />
4. Click \*\*Add Contact \*\*to add the contact.

Once saved, the contact will appear in the Contacts table associated with the current entity.

## Adding an Existing Contact

If a contact already exists in your system (e.g. previously added to another policy, insured, or claim), you can quickly associate them with this entity using the **Existing Contact** option:

#### Steps to Add an Existing Contact

1. Navigate to the entity’s **Contacts** tab (e.g. for an insured, policy, or claim).
2. Click the **Add Contact** button in the top-right of the Contacts table.
3. From the dropdown, select **Existing Contact**.

   <img
     src="/images/Screenshot2025-07-14at3.05.00PM.png"
     alt="Screenshot2025 07 14at3 05 00PM Pn"
     title="Screenshot2025 07 14at3 05 00PM Pn"
     style={{ width:"95%" }}
   />
4. In the **Add Existing Contacts** modal:
   - Use the dropdown to search for an existing contact by name.
   - Select the desired contact(s).
5. Click **Add Contacts** to associate the selected contact(s) with this entity.

<img
  src="/images/Screenshot2025-07-14at4.04.20PM.png"
  alt="Screenshot 2025-07-14 at 4.04.20 PM.png"
  title="Screenshot 2025-07-14 at 4.04.20 PM.png"
  style={{ width:"73%" }}
/>

The selected contacts will now appear in the Contacts table and be associated with this entity
